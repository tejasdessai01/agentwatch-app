Quick Start Guide

1. Install the SDK

Run this in your agent's project folder:

npm install clawsight-client

2. Add to Your Agent

Copy this code into your agent's main file:

const monitor = require('clawsight-client');
const watcher = monitor({
  server: 'https://agentwatch-dashboard.onrender.com',
  token: 'YOUR_API_KEY_HERE'
});

// Register Agent
watcher.init({
  id: 'my-agent-01',
  name: 'Production Bot'
});

// Send Logs & Metrics
watcher.log('Task started...', 'working');
watcher.metric('cost', 0.05);
